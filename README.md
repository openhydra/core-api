# core-api

* core-api 项目是 openhydra-2.0 版本的核心 api 项目，他主要将前端和后端的交互进行了封装，提供了一些基础的功能，比如用户登录，用户注册权限管理等等。
* openhydra 致力于提供在线的稳定的 ai 教育领域的平台服务，他可以快速的在任何 k8s 环境中部署
